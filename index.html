Estrutura Básica do HTML

(Voltar)

A tag < html>> define que tudo o que estiver entre ela faz parte do documento html.
Essa tag < head> significa cabeçalho nela estão informações sobre nosso documento html, como a tag < title> se refere ao título que será exibido na aba do nosso navegador. Essas informações não são visíveis para o cliente.
Dentro da tag < body> temos tudo o que vai aparecer para o usuário, é onde o conteúdo da nossa página será inserido.

Falando Sobre Tags

(Voltar)

Tag < i> essa tag deixa em itálico o que estiver entre ela.
Tag < strong> formata o caractere em negrito
Tag < /br> usada como quebra de linha.

Algumas Tags não possuem uma tag de fechamento, uma delas por exemplo é a tag < input> que por si só gera um campo para digitação.

Outro exemplo é a tag < img> que aceita argumentos/atributos, mas não possui uma tag de fechamento.
Atributos são uma forma de configurar para que o elemento seja exibido de uma forma personalizada.

Atributos das Tags

(Voltar)

Como dito antes sobre os atributos propriedades que as tags podem ter alguns desses atributos são globais, ou seja, servem para a maioria das tags, outros são específicos a cada tag.
Propriedades/Atributos Globais:

  • As tags < p> e < h1> entre outras, podem conter o atributo global "id", que serve como um identificador deste elemento. No javascript, é possível alterar as propriedades de um elemento ou grupo através desse atributo.
  • Atributo style, usado no CSS interno, externo e inline, para modificar o estilo de “apresentação” de um elemento.
  • Atributo class = " ": com função semelhante ao id o class pode ser usado para padronizar o estilo de um grupo de elemento com essa tag.
  • O elemento < input> muito usado em formulários, possui o atributo type específico, que pode variar de text, date entre outros, esse type quando dentro do elemento < input> pode assumir diversas funções.
  • type = “number” (cria um campo com valores numéricos)
  • type = “color” (cria um campo com um seletor de cores)
  • A tag < img> recebe como atributo o src = " " de source onde entre aspas é inserido o caminho do arquivo ou link onde está a imagem que desejamos inserir em nossa página.

  • O elemento < img> também pode receber o atributo width = “150” usado para definir largura, altura, comprimento da imagem selecionada. < img width ="300" src= "img/ex1.png">

As tags responsáveis pela exibição de títulos são < h1> < /h1> que varia em níveis até o < h6>< /h6> onde h1 é um nível mais alto o h2 é um subtítulo e assim por diante. O h dessa tag significa header/cabeçalho;

    Exemplo:
  • < h1>Formatando Textos com HTML< /h1>
    < h2>Aula 06< /h2>
    < h3>DIO.me< /h3>


  • Entre as tags úteis para a formatação de um texto em html, temos a tag < mark>< /mark> inserida quando desejamos marcar de forma colorida uma frase, palavra ou texto.

Listas Ordenadas e Não Ordenadas

(Voltar)

Listas Ordenadas seguem a seguinte sintaxe:


A tag < ol> define uma lista ordenada. Listas ordenadas podem ser numericas ou alfabéticas. Use a tag < li> de list item para definir cada item da lista.
Listas ordenadas podem ainda conter o atributo start = "" usado para definir o início de uma listagem. E também o atributo type = "" para definir o tipo de lista.
Exemplo: Lista Ordenada Numérica
  1. Café
  2. Chá
  3. Leite


Exemplo: Lista Ordenada Alfabética
  1. Café
  2. Chá
  3. Leite

Listas Não Ordenadas seguem a seguinte sintaxe:


A tag < ul> define uma lista não ordenada. Use a tag < li> de list item para definir cada item da lista.
Listas ordenadas podem ainda conter o atributo style = "" para definir o tipo de marcação da lista.
Exemplo: Lista Não Ordenada style ="square"
  • Café
  • Chá
  • Leite

Falando Sobre Links

(Voltar)
A tag < a> define a marcação de um hyperlink, usada para linkar de uma página para outra.
O atributo mais importante do elemento < a> é o href, o qual indica o destino do link. Por padrão, os links aparecem da sequinte forma em todos os navegadores:
  • Um link não visitado é sublinhado e azul.
  • Se o link já foi visitado ele é sublinhado e roxo.
  • Um link ativo é sublinhado e vemelho quando clicado.
A tag de link < a> pode ser acompanhada do atributo target = "_blank" . Com a função de redirecionar o link para ser aberto em outra aba/página. Outro atributo complementar é o title = " ", tem a função de adicionar um informativo sobre o link que será usado assim que o cursor estiver em cima.
Exemplo: Mais Sobre HTML

OBS! - Outra característica dos links é que além de linkar páginas externas dentro do projeto, podem também servir como ferramenta de navegação no site, fazendo uma ancoragem e ligando tópicos a parágrafos dentro da página por exemplo.
